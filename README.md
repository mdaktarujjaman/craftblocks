# CraftBlocks

**Modern Gutenberg Blocks Library for WordPress**

CraftBlocks is a custom-built collection of professional, production-ready Gutenberg blocks — designed with clean architecture, full customization control, and performance in mind. Built from scratch using PSR-4 autoloading, OOP principles, and the native WordPress Block Editor APIs.

---

## ✨ Features

### Blocks (Phase 1 — In Progress)
- ⏳ **Hero Banner** — Heading, subheading, button, background image, overlay, alignment
- ⏳ **Pricing Table** — Price, currency, feature list, button, featured badge
- ⏳ **FAQ Accordion** — Unlimited questions, rich text, collapsible, icons
- ⏳ **Team Members** — Image, name, position, bio, social links
- ⏳ **Call To Action** — Title, description, button, background, gradient

### Blocks (Phase 2 — Planned)
- 🔲 Notice
- 🔲 Feature List
- 🔲 Timeline
- 🔲 Testimonials
- 🔲 Stats Counter

### Core Capabilities (Planned across all blocks)
- 🎛️ **Inspector Controls** — Padding, margin, border radius, shadow, typography, responsive settings
- 🎨 **Color Support** — Background, text, button, hover states
- 📱 **Responsive Design** — Desktop, tablet, and mobile breakpoints
- 📦 **Dedicated Block Category** — All blocks grouped under "CraftBlocks" in the inserter

## 🛠️ Tech Stack

- **PHP 8.1+** — OOP, PSR-4 autoloading via Composer
- **React** (`@wordpress/scripts`) — Block editor UI
- **WordPress Block API v3** — Modern block registration via `block.json`
- **SCSS** — Block styling

## 🚀 Installation

### For users
1. Download the latest release from the [Releases](https://github.com/mdaktarujjaman/craftblocks/releases) page
2. Upload the `craftblocks` folder to `/wp-content/plugins/`
3. Activate **CraftBlocks** from the WordPress Plugins screen

### For developers
```bash
git clone https://github.com/mdaktarujjaman/craftblocks.git
cd craftblocks

# Install PHP dependencies
composer install

# Install JS dependencies
npm install

# Build blocks for production
npm run build

# Or start development mode with live rebuild
npm run start
```

## 🏗️ Architecture

craftblocks/
├── craftblocks.php          # Plugin bootstrap
├── includes/                 # PHP backend (PSR-4: CraftBlocks)
│   ├── Core/                 # Plugin lifecycle, hook loader
│   ├── Blocks/                # Block auto-registration
│   └── Support/               # Block category, shared utilities
├── src/                      # React block source (per block)
│   └── hero-banner/
│       ├── block.json
│       ├── index.js
│       ├── edit.js
│       ├── save.js
│       └── style.scss
└── build/                    # Compiled output (generated by wp-scripts)

Each block is auto-discovered and registered from the `build/` directory — no manual registration required when adding new blocks.
